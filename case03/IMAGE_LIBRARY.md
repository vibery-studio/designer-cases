# PORTFOLIO IMAGE LIBRARY

Danh sách hình ảnh ổn định để sử dụng trong các portfolio templates.
Tất cả URLs đều từ Unsplash với format cố định, có thể điều chỉnh kích thước.

---

## CÁCH SỬ DỤNG

**Format URL Unsplash:**
```
https://images.unsplash.com/photo-[ID]?w=[WIDTH]&h=[HEIGHT]&fit=crop&crop=[FOCUS]
```

**Ví dụ:**
```html
<img src="https://images.unsplash.com/photo-1573496359142-b8d87734a5a2?w=600&h=800&fit=crop&crop=face" alt="Portrait">
```

**Crop options:** `face`, `center`, `top`, `bottom`, `left`, `right`

1. **Tất cả URLs đều hoạt động** - đã verify

4. **fit=crop** để đảm bảo fill container
